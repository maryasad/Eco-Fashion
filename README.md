# EcoTextile - Sustainable Fashion & Textile Recycling Platform

A modern web platform promoting textile recycling and sustainable fashion practices.

## Features

- Educational content about textile recycling
- Step-by-step recycling guides
- Interactive tools and calculators
- Community forum
- Sustainable fashion resources
- Environmental impact tracking

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Visit `http://localhost:3000` in your browser
